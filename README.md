The current project is intended for Android use (such as Android Studio), but can also be used on website. If at any time running the website implementation is required to be used, please open Ctt/lib/pages/utils/base_url.dart and change the baseUrl, commenting the mobile app url (10.0.2.2) and uncomment the web url (localhost), this step is mandatory since the backend will differ when connecting to the database.


To run the project, please open two command prompt at root ctt, the first one:

cd backend

npm install

npm start


The second one:

flutter run
